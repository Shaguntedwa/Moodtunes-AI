\# 🎧 MoodTunes AI

\### Music for every emotion



MoodTunes AI is an empathetic music companion that transforms your current mood into a personalized playlist, complete with a supportive message, motivational quote, and self-care suggestion — powered by Gemini AI.



\---



\## ✨ Features



\- 🎵 Personalized song recommendations based on mood + genre

\- 💌 Supportive messages tailored to how you're feeling

\- ✨ Motivational quotes to uplift you

\- 🌱 Self-care activity suggestions

\- 🔗 Direct Spotify search links for every song

\- 📥 Download your playlist as a text file

\- 📊 Mood history tracker (within session)

\- 🎭 Music Personality Badge based on your mood



\---



\## 🛠 Tech Stack



\- Python

\- Streamlit

\- Google Gemini AI (gemini-2.5-flash-lite)



\---



\## 🚀 Run Locally



\*\*1. Clone the repo\*\*

```bash

git clone https://github.com/YOUR\_USERNAME/YOUR\_REPO\_NAME.git

cd YOUR\_REPO\_NAME

```



\*\*2. Install dependencies\*\*

```bash

pip install -r requirements.txt

```



\*\*3. Set up your API key\*\*



Create a `.streamlit/secrets.toml` file:

```toml

GOOGLE\_API\_KEY = "your-gemini-api-key-here"

```

Get your free key at: https://aistudio.google.com



\*\*4. Run the app\*\*

```bash

streamlit run app.py

```
